Review Notes

Purpose: users sign in to our web application using their TikTok account.

Scope usage: user.info.basic is requested only to identify the user after consent and to retrieve basic profile information needed for authentication.

Expected review flow: open this page, click the TikTok login button, approve access on TikTok, and return to the callback page where the authorization response is displayed.
