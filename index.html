1. Start TikTok OAuth

Enter the TikTok app settings. These values stay only in this browser and are used to build the authorization URL. After TikTok redirects back, the callback page stores the authorization code for the posting demo below.

2. Review Notes

Purpose: authorized users sign in with TikTok, then prepare and submit a content posting request for sandbox/review demonstration.

Requested scopes: basic user identity for login and the posting scope needed for authorized content publishing.

Expected review flow: open this page, start TikTok authorization, approve access, return to the callback page, exchange the code for tokens, then submit a content-posting request from this site.

5. Demo checklist for review recording

  • Open this site on the registered review domain.
  • Show the TikTok app settings entered into the OAuth panel.
  • Start OAuth and approve the app in TikTok sandbox/review mode.
  • Show that the callback page receives the authorization code.
  • Return here and exchange the code for tokens.
  • Fill in a demo video URL, caption, and privacy settings.
  • Submit the content posting request and show the response on screen.
Important: This static site is meant for review/demo visibility. For production, secret handling and token exchange should be moved to a backend service instead of running in the browser.
